# 🌍 Worldwide Fast Delivery

A modern, responsive package tracking website for global shipping services.

## 📋 Features

- **🏠 Home Page**: Showcase of services and features
- **📦 Package Tracker**: Real-time package tracking with timeline
- **ℹ️ About Page**: Company mission, values, and benefits
- **📧 Contact Page**: Contact form and support information
- **🎨 Modern Design**: Responsive, animated UI with gradient colors
- **⚡ Fast Loading**: Optimized performance
- **📱 Mobile Friendly**: Works perfectly on all devices

## 🚀 Live Demo

Visit the live site: [Worldwide Fast Delivery](https://zeddgfdx-commits.github.io/worldwidefastdelivery/)

For testing the tracker, use these sample tracking numbers:

- **WFD123456789** - Delivered package
- **WFD987654321** - Out for delivery
- **WFD555555555** - In transit

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Flexbox, Grid, Animations, Gradients
- **JavaScript** - DOM manipulation, Event handling
- **Responsive Design** - Mobile-first approach

## 🎨 Design Highlights

- **Color Scheme**: Purple gradient (#667eea to #764ba2)
- **Animations**: Smooth fade-in, slide-up effects
- **Typography**: Segoe UI with proper hierarchy
- **Spacing**: Consistent padding and margins
- **Interactivity**: Hover effects and transitions

## 📱 Responsive Breakpoints

- Desktop: Full layout
- Tablet: Adjusted grid and spacing
- Mobile: Stacked layout, optimized touch targets

## 🔄 How to Use the Tracker

1. Navigate to the "Track Package" page
2. Enter a valid tracking number (e.g., WFD123456789)
3. Click "Track" or press Enter
4. View the package status and timeline
